# Minions Match

Match scoring, shortlists, and hard gate evaluation between jobs and profile

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-match/sdk`) — Core library for Node.js
- **Python SDK** (`minions-match`) — Python bindings
- **CLI Tool** (`@minions-match/cli`) — Command-line interface
- **Web App** — Interactive playground at [match.minions.wtf](https://match.minions.wtf)
