---
title: Installation
description: Install Minions Match packages
---

## TypeScript / Node.js

```bash
npm install @minions-match/sdk minions-sdk
```

## Python

```bash
pip install minions-match
```

## CLI

```bash
npm install -g @minions-match/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
