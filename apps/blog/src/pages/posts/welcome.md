# Welcome to Minions Match

We're excited to introduce **Minions Match** — Match scoring, shortlists, and hard gate evaluation between jobs and profile.

## What's Included

- **TypeScript SDK**: `npm install @minions-match/sdk`
- **Python SDK**: `pip install minions-match`
- **CLI**: `npm install -g @minions-match/cli`

## Getting Started

Check out the [documentation](https://match.minions.help) for a full guide.
